MinOS

C Assembly Operating Systems

A minimalistic operating system built from scratch, focusing on core OS concepts and bare metal programming.

Key Components:

  • Bootloader: Custom two-stage bootloader handling system initialization and kernel loading
  • Memory Management: Implementation of paging, virtual memory system, and basic memory allocator
  • Process Management: Basic process scheduling and context switching
  • Interrupt Handling: Custom IDT (Interrupt Descriptor Table) and ISR (Interrupt Service Routines)
  • Device Drivers: Basic drivers for keyboard, display, and timer
  • File System: Simple file system implementation for basic storage operations

BitTorrent Client

P2P Kademlia Networking

A pure peer-to-peer BitTorrent client implementation with distributed hash table (DHT) based on the Kademlia protocol.

Key Components:

  • Kademlia DHT: Implementation of distributed hash table for peer discovery and network organization
  • P2P Protocol: Pure peer-to-peer architecture without central trackers
  • Node Management: K-bucket implementation for efficient peer routing and lookup
  • File Handling: Piece management, verification, and parallel downloading
  • Network Layer: UDP for DHT operations, TCP for peer data transfer
  • BitTorrent Protocol: Support for standard BitTorrent protocol messages and extensions

AlphaGomoku

Deep RL PyTorch MCTS Neural Networks

A deep reinforcement learning system designed to master the game of Gomoku (Five in a Row), inspired by DeepMind's AlphaGo architecture.

Key Components:

  • Neural Network Architecture: Dual policy and value networks for move prediction and position evaluation
  • Monte Carlo Tree Search: Advanced MCTS implementation guided by neural networks
  • Self-Play Training: Iterative self-play mechanism for continuous improvement
  • Policy Optimization: Implementation of policy gradient methods with value bootstrapping
  • Distributed Training: Parallel self-play and network training pipeline
  • Position Evaluation: Board state evaluation using convolutional networks
